# System Use Cases

| Use Case ID | Use Case Name | Actors Involved | Description |
| :--- | :--- | :--- | :--- |
| **UC-U01** | **Authenticate (Register/Login)** | Guest, Customer | Register a new account or log in using email or social accounts (Google/Facebook). |
| **UC-U02** | **Forgot/Reset Password** | Guest, Customer | Request a password reset link and set a new password. |
| **UC-U03** | **Browse Products** | Guest, Customer | Browse the perfume catalog and view available product listings. |
| **UC-U04** | **Search & Filter** | Guest, Customer | Search perfumes by keywords and apply filters like brand, price, or scent family. |
| **UC-U05** | **View Product Details** | Guest, Customer | View detailed information, scent notes, and descriptions of a specific perfume. |
| **UC-U06** | **View Content & Promotions** | Guest, Customer | View public promotions, journal articles, and physical store locations. |
| **UC-C01** | **Manage Profile** | Customer | Update personal profile information and preferred scent types. |
| **UC-C02** | **Manage Shipping Addresses** | Customer | Add, update, or remove delivery addresses for orders. |
| **UC-C03** | **Manage Cart** | Customer | Add, update quantity, or remove products from the shopping cart, and apply promotion codes. |
| **UC-C04** | **Place Order** | Customer | Submit an order and choose online payment methods (VNPay/Momo). |
| **UC-C05** | **Confirm COD Order** | Customer | Confirm cash-on-delivery order during checkout. |
| **UC-C06** | **View Purchase History** | Customer | View the list of previous orders and purchases. |
| **UC-C07** | **View Order Details** | Customer | View detailed purchase history and specific order items. |
| **UC-C08** | **Track Order Status** | Customer | Track the delivery and processing status of active orders. |
| **UC-C09** | **Cancel Order** | Customer | Cancel a pending order before it is processed. |
| **UC-C10** | **Request Order Return** | Customer | Submit and track product return or refund requests. |
| **UC-C11** | **Interact with AI Chatbot** | Customer | Chat naturally with the AI consultant to receive perfume advice. |
| **UC-C12** | **Take AI Quiz** | Customer | Complete the AI-guided perfume quiz to refine scent preferences. |
| **UC-C13** | **Receive AI Recommendation** | Customer | Receive and view personalized perfume recommendations generated by AI. |
| **UC-C14** | **Manage Favorites** | Customer | Add or remove perfumes from the favorites list. |
| **UC-C15** | **Write/Edit Product Review** | Customer | Submit or edit a review and rating for purchased products. |
| **UC-C16** | **View/Manage Notifications** | Customer | View, read, and delete in-app system notifications. |
| **UC-C17** | **View Loyalty Points** | Customer | View accumulated loyalty points from purchases. |
| **UC-S01** | **Perform In-store POS Sale** | Staff | Process in-store sales by scanning barcodes, applying discounts, and printing receipts. |
| **UC-S02** | **Manage Store Inventory** | Staff | Check current inventory status and update stock quantities after sales or restocks. |
| **UC-S03** | **Process Return/Exchange** | Staff | Handle product returns or exchanges directly at the physical store. |
| **UC-S04** | **Assist Customer with AI** | Staff | Assist in-store customers using the AI consultation tool. |
| **UC-S05** | **View Daily Sales Summary** | Staff | View the daily sales performance and summary for the store. |
| **UC-A01** | **Manage Users & Roles** | Admin | View, update, ban user accounts, and assign roles (Customer, Staff, Admin). |
| **UC-A02** | **Manage Product Catalog** | Admin | Add, update, or remove perfume products, brands, categories, and scent attributes. |
| **UC-A03** | **Manage Inventory & Alerts** | Admin | Import inventory data in bulk and configure stock and expiry alerts. |
| **UC-A04** | **Manage Orders & Returns** | Admin | View all system orders, update order statuses, and process customer return and refund requests. |
| **UC-A05** | **Manage Marketing & Loyalty** | Admin | Create promotion campaigns and manage the loyalty rewards program. |
| **UC-A06** | **Manage Banners/Sliders** | Admin | Create, update, or remove homepage promotional banners and sliders. |
| **UC-A07** | **Manage Journals/Blogs** | Admin | Write, edit, publish, or delete blog posts and journal articles. |
| **UC-A08** | **Manage Product Reviews** | Admin | Moderate, approve, or hide customer reviews. |
| **UC-A09** | **Manage Physical Stores** | Admin | Create store locations and assign staff members to specific stores. |
| **UC-A10** | **Configure System Settings** | Admin | Configure AI recommendation parameters and shipping fee rules for GHN/GHTK. |
| **UC-A11** | **View Analytics Dashboard** | Admin | View comprehensive system analytics and performance reports. |
| **UC-A12** | **Backup & Restore Data** | Admin | Perform system data backups and data recovery operations. |
| **UC-X01** | **Authenticate via OAuth** | OAuth Provider | Authenticate users via Google or Facebook providers. |
| **UC-X02** | **Process Payment Transaction** | Payment Gateway | Process online transactions and handle payment confirmation webhooks. |
| **UC-X03** | **Request AI Recommendation** | AI Service | Send requests for natural language AI-based perfume recommendations. |
| **UC-X04** | **Process Shipping Operations** | Shipping Provider | Calculate shipping fees and sync delivery orders via GHN/GHTK API. |
| **UC-X05** | **Send Notification** | Email Service | Dispatch system emails or SMS notifications to users. |
